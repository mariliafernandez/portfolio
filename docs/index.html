projects i'm working on

Now I'm focused on my thesis. I'm proposing an image descriptor based on Hilbert's curve. A descriptor is a vector that hold information about features of an image. Once we describe a feature, we're able to detect it. So the goal is to improve the process of object recognition by testing this new approach.

I'm also studying Computer Graphics with OpenGL, in Python. For now I've worked on a code to render a .obj file and add options to translate, scale and rotate the object. The code for this project is available here. And if you're interested in learning about OpenGL, here are some useful links: Learn OpenGL and The Cherno
